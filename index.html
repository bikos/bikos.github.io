Traditional Method

This method utilizes the method proposed by Micheal Whitman which makes the calculation using the asset value, likelihood, control and the uncertainity property of the asset.
The calculation is done as:

(Asset x Risk) - (Asset x Risk) x Control + (Asset x Risk) x Uncertainity
